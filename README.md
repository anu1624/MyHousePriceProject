# House Price Predictor – Mumbai

A Java project that predicts house prices in Mumbai using a MySQL database.  
This project demonstrates data-driven prediction with a dataset of house features, integrating SQL for data management and Java for building a user-friendly interface.

---

## Features
- Predicts house prices based on input features like area, bedrooms, location, etc.
- Uses MySQL database to store and manage housing data.
- Interactive Java GUI for easy user input and display of predicted price.
- Lightweight and easy to extend for additional features.
